# QHAD (Qibla Hilal Automation Direction) 🛰️🏆

> 🥇 **Pemenang Juara 1 Lomba Inovasi Falak DI OASE PTKIN SE-INDONESIA II**

**QHAD** adalah sistem otomasi robotik untuk penentuan arah objek langit secara presisi. Alat ini tidak hanya menunjuk kiblat, tetapi mampu bergerak otomatis mengikuti posisi Matahari, Bulan, dan Hilal menggunakan sinkronisasi data online.

## 🌟 Keunggulan Utama
* **Automated Tracking:** Menggerakkan motor secara otomatis ke posisi koordinat objek langit.
* **IoT Enabled:** Menggunakan modul WiFi untuk mengambil data kalkulasi ephemeris dari mesin komputasi [panonpoe.id](https://panonpoe.id).
* **High Precision:** Implementasi algoritma Jean Meeus untuk akurasi data astronomi tingkat tinggi.

## 🛠️ Teknologi & Sistem
* **Logic:** Python & C++.
* **Connectivity:** WiFi IoT (Real-time Web Sync).
* **Hardware:** Precision Stepper/Servo Motors, Arduino/ESP Controller.
